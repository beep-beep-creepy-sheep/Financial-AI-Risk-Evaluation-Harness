# Data

All data in this directory is synthetic and generated locally by:

```bash
fin-ai-risk-eval generate-data
```

Files:

- `synthetic_credit_data.csv`: synthetic tabular credit-risk records.
- `eval_cases.csv`: synthetic consumer-finance evaluation prompts.
- `risk_taxonomy.json`: structured risk taxonomy used by the project.

No real customer data, real bank data, paid datasets, or personal information is used.
